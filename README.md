# Книгозакладчик. BookCrossing_Project

: book: Сервис объединяющий людей и дарящий книгам свободу

Для установки библиотек:

```
npm i
```

### Running

Для запуска в папке client:

```
yarn start
```

Для запуска в папке server:

```
npm start
```

Доступ http://localhost:3000/

### Возможности сервиса:

- Поделиться книгой.
Ввести необходимую информацию, выбрать точку на карте и вписать уникальный номер в книгу.
- Взять книгу.
Выбрать книгу на карте и бесплатно стать ее новым временным владельцем, указав ее уникальный номер
- Передать книгу дальше
Позволить книге путешествовать дальше выбрав для нее новый временный адрес
- Обсудить книгу.
Пообщаться на тему книги с теми к кому она попадала в руки и возможно найти новых друзей
- Следить за книгой.
Отпустив книгу в путешествие следить за ее перемещениями не выходя из дома!

### Функциональность

- Регистрация и вход в систему
- Получение уникального идентификатора для книги
- Карта с отметками закладок книг
- Страница пользователя 
- Страница книги с возможностью общения между пользователями
